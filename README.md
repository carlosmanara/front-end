# ONG Amigo Fiel

Site institucional de proteção animal desenvolvido em 3 etapas.

## Atividades

1. **HTML** - Estrutura semântica
2. **HTML + CSS** - Design responsivo
3. **JavaScript** - SPA com contador animado e máscaras

## Stack

HTML5 • CSS3 • JavaScript • IMask.js

## Como visualizar

Clone e abra o `index.html` da atividade desejada.
