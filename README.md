# ONG Amigo Fiel - Proteção Animal

Projeto desenvolvido para o curso de Análise e Desenvolvimento de Sistemas, simulando um site institucional para uma ONG fictícia de proteção animal.

## Estrutura do Projeto

O repositório contém 3 atividades progressivas, a que se encontra na raiz do repositório é a atividade 3:

### **Atividade 1 - HTML Puro**
- Estrutura HTML5 semântica
- 4 páginas: Home, Cadastro, Projetos e Quem Somos
- Foco em acessibilidade e boas práticas

### **Atividade 2 - HTML + CSS**
- Mesma estrutura da Atividade 1
- Estilização completa com CSS3
- Layout responsivo (mobile, tablet, desktop)
- Design moderno e profissional

### **Atividade 3 - JavaScript (SPA)**
- Single Page Application
- Contador animado de conquistas
- Máscaras de input (CPF, telefone, CEP)
- Sistema de rotas simples
- Navegação sem reload

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla)
- IMask.js (máscaras de input)

## Como Executar

1. Clone o repositório:
```bash
git clone https://github.com/carlosmanara/front-end.git
```

2. Abra o arquivo `index.html`. A forma mais recomendada é usar a extensão Live Server no Visual Studio Code para ter uma simulação de ambiente real e recarregamento automático.
