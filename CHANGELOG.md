# Changelog

As mudanças neste projeto serão documentadas aqui.

## [4.0.0] - 08/11/2025

### Adicionado
- Acessibilidade WCAG 2.1 Nível AA
- Modo escuro responsivo
- Navegação por teclado completa
- ARIA labels em elementos interativos
- Arquivos minificados para produção
- Imagens otimizadas

### Melhorado
- Documentação do README
- Estrutura de commits
- Organização do repositório

## [3.0.0] - 03/11/2025

### Adicionado
- Single Page Application (SPA)
- Contador animado com Intersection Observer
- Máscaras de input (CPF, telefone, CEP)
- Sistema de rotas com hash

## [2.0.0] - 01/11/2025

### Adicionado
- Estilização completa com CSS3
- Design responsivo (mobile, tablet, desktop)
- Grid e Flexbox layouts
- Animações e transições

## [1.0.0] - 28/10/2025

### Adicionado
- Estrutura HTML5 semântica
- 4 páginas: Home, Cadastro, Projetos, Quem Somos
- Formulário de cadastro
- Sistema de navegação
